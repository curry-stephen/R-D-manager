按以下步骤进入目录：
yiyuan->src

assets目录：
	images目录、img目录为前端图片素材
logo：项目logo

components目录：

		Header：信息管理系统头部组件

echartsvue目录：
		BinTu：门户网站饼图组件
		GradTu：门户网站仪表盘组件
		LeiDaTu：门户网站雷达图组件
		LouDou：门户网站漏斗图组件
		Workecharts：门户网站柱状图组件
		Zhexian：门户网站堆叠图组件

layout目录：
	LayOut：信息管理系统框架组件

menhucomponets目录：
			public目录：
				Footer：门户网站底部组件
				WebHeader：门户网站头部组件
			
			FuWuLiLian：门户网站首页页面组件
			MyFuwu：门户网站首页页面组件
			DrugAndFuItem：门户网站医院网站页面组件
			DongtaiImg：门户网站动态分享页面组件
			NowData：门户网站动态分享页面组件
			HotNews：门户网站动态分享页面组件

			Works：门户网站动态分享页面组件
			NowDataItem：门户网站动态分享页面组件

menhuview目录：
		DataCenter：门户网站数据中心页面
		DongtaiAbout：门户网站动态分享页面
		DrugAndFu：门户网站医院党建页面
		WebHome：门户网站首页页面

router目录：
	index：前端路由配置

utils目录：
	mhrequst：门户网站数据请求封装
	request：信息管理数据请求封装

views目录：
	Bill：信息管理系统账单信息页面
	Druginfo：信息管理系统药品信息页面
	Inspect：信息管理系统保质期查询页面

	Matter：信息管理系统问题药品页面
	Owinfo：信息管理系统药品出入库页面
	Rgoods：信息管理系统退货页面
	Sale：信息管理系统销售记录页面
	Supplyinfo：信息管理系统供应商信息页面
	Tosupply：信息管理系统退货给供应商页面
	Updatepwd：信息管理系统个人信息页面
	UserLog：信息管理系统用户操作日志组件
	UserMapper：信息管理系统用户管理页面

main：前端全局配置文件
